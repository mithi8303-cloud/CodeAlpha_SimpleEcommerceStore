# CodeAlpha Simple E-Commerce Store

## Project Overview

This is a Full Stack E-Commerce Store developed using HTML, CSS, JavaScript, Node.js, and Express.js.

## Features

* User Registration
* User Login
* Product Listing
* Add to Cart
* Remove from Cart
* Checkout Functionality
* Order History

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Postman

## API Endpoints

* GET /products
* POST /register
* POST /login
* POST /cart
* GET /cart
* DELETE /cart/:id
* POST /checkout
* GET /orders
